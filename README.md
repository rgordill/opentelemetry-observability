# OpenTelemetry Observability

This repository contains related resources about OpenTelemetry Collector and its components.

## Contents

### HostMetrics Receiver

The [HostMetricsReceiver](HostMetricsReceiver/) directory contains comprehensive resources for the OpenTelemetry Collector's `hostmetrics` receiver, including:

- Example configurations for collecting system-level metrics
- Grafana dashboard for visualizing host metrics
- Detailed metric comparison with Node Exporter
- Documentation and usage examples

See the [HostMetricsReceiver README](HostMetricsReceiver/README.md) for complete documentation.
